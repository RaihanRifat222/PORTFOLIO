# Agentic IT Support System — Azure AI + AutoGen

## Overview
Designed and implemented an agentic AI system for automated IT issue resolution using AutoGen and Azure AI services.

## Architecture
- Multi-agent collaboration framework
- Issue classification agent
- Semantic retrieval agent
- Solution generation agent
- Vector-based knowledge base

## Key Features
- Retrieval of historical IT tickets
- Retrieval-grounded solution generation
- Azure AI service integration

## Skills Demonstrated
- Agent orchestration
- Azure AI integration
- Semantic retrieval systems
