# SmartCourse — Agentic AI Curriculum Builder

## Overview
SmartCourse is a full-stack agentic AI system that generates structured course curricula and module content from user-defined learning requests.

## Architecture
- FastAPI backend
- React frontend
- Multi-agent orchestration using AutoGen
- JSON schema validation loops
- Persistent JSON-based storage

## Agent Design
- Curriculum Architect Agent
- Validator Agent
- Module Author Agent

The system ensures structured and schema-compliant outputs through regeneration loops when validation fails.

## Features
- Course generation endpoint
- Module regeneration
- Browse previously created courses
- Drill-down module interface

## Technical Highlights
- Modular generation pipeline
- Structured JSON validation enforcement
- Frontend-backend API integration
- Live deployment
