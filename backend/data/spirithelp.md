# SpiritHelp — Retrieval-Grounded LLM System

## Overview
SpiritHelp is a retrieval-augmented conversational AI system designed to study hallucination reduction and responsible LLM behavior.

## Architecture
- LangChain pipeline
- ChromaDB vector database
- GPT-4 response generation
- Hugging Face Spaces deployment

## Key Features
- Retrieval-grounded generation
- Hallucination reduction constraints
- Semantic similarity search
- Public demo deployment

## Skills Demonstrated
- RAG pipeline design
- Vector database integration
- Prompt engineering
- Deployment to Hugging Face
